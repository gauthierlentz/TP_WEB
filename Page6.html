Les tableaux à une dimension (vecteurs) :


Syntaxe : Type nom_de_tableaux=Tableau[1..N] de TypeElt

Les tableaux à deux dimensions :


Syntaxe : Type nom_du_tableau=Tableau[1..N,1..N] de TypeElt

Exemple 1


Algorithme Tableau
Const
MAX : ENTIER <-50
VAR
T1 : Tableau [1..MAX] de CHAINE
T2 : Tableau[1..5,1..10 de ENTIER
DEBUT
T1[1]<-‘a’
ECRIRE T1[1]
T2[2,7]<-12
ECRIRE T2[2,7]
FIN

_________________________________________________________________________________
Algorithme INITTAB
CONST
MAX :ENTIER<-50
Procedure InitTabT1(T1 :Tableau[1..MAX] de ENTIER)
VAR
I :ENTIER
DEBUT
POUR i de 1 A MAX FAIRE
T1[i]<-0
FINPOUR
FINPROCEDURE


Procedure initTabT2(T2 :Tableau[1..5,1..10] de ENTIER)
VAR
I : ENTIER
J : ENTIER
DEBUT
POUR i de 1 A 5 FAIRE
POUR j de 1 A 10 FAIRE
T2[i,j]<-0
FINPOUR
FINPOUR
FINPROCEDURE


//Programme principal
VAR
T1 : Tableau[1..MAX] de ENTIER
T2 : Tableau[1..5,1..10] de ENTIER
DEBUT
InitTabT1(T1)
InitTabT2(T2)
FIN
