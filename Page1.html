Quand on parle de variable on parle obligatoirement de :

  • Déclaration : Une variable doit toujours être déclarée
  • Affectation : Le nombre qu’on donne à cette variable
  • Utilisation
  • Réaffectation : quand j’appelle ma variable qui a un état initial dans laquelle je rajoute une valeur

    • Les types élémentaires possibles sont :

  • Les étiers
  • Les réels
  • Les caractères
  • Les chaines de caractères
  • Les booléens

    • Ecriture variable :


    NomDeLaVariable <- Valeur : TYPE
    (Exemple : Nb3 <- Nb1-Nb2 x 4 : REEL)

    Ecriture constante :


    NomDeLaConstante <- Valeur : Type
    (Exemple : Pi <- 3.14 : REEL)

    Notion de données :

    Les données manipulées par les instructions sont :
  • Des variables proprement dites
  • Des constantes
  • Des valeurs littérales (« Zombie U », 45, VRAI)
  • Des expressions complexes (combinaison de variables, constantes et valeurs littérales avec des opérateurs

    Primitives :

    Une instruction est un ordre élémentaire au sens algorithmique, que peut exécuter un programme.
    Deux instructions d’entrée-sortie :

  • LIRE : lecture de la frappe (au clavier)
  • ECRIRE : affichage (à l’écran)
